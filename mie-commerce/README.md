Proyecto E-Commerce

Full Shop - Tienda Online

Descripción
Aplicación web de e-commerce desarrollada con React. Permite visualizar productos,
ver detalles, agregar al carrito y realizar compras.

Tecnologías Utilizadas

- React
- React Router
- Firebase Firestore
- Context API
- CSS / SASS

Funcionalidades

- Catálogo de productos
- Detalle de producto
- Carrito persistente
- Checkout con generación de orden
- Almacenamiento en Firestore
